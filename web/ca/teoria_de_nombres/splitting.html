Splitting d’algebres de quaternions

Una àlgebra de quaternions B/F és split si B ≅ M2(F).

Un cos K que conté F és un splitting field per B, si B⊗𝔽K és split.

Exemple: ℍ⊗ℝℂ ≅ M2(ℂ)

1 ⊗ 1 ↦ id 

1 ⊗ i ↦ id  (1⊗i)2 = 1 ⊗ (−1)

i ⊗ 1 ↦ id  (i⊗1)2 = (−1⊗1)

(1⊗i+i⊗1)2 = 1 ⊗ (−1) + 2i ⊗ i + (−1) ⊗ 1

i ⊗ i ↦ id  j ⊗ 1 ↦ id  j ⊗ i ↦ id  k ⊗ 1 ↦ id  k ⊗ i ↦ id
